# Check Status

Check status, syntax, or state of specified item.

- **Service**: `curl http://localhost:PORT/api/health`
- **Git**: `git status && git log --oneline -5`
- **Syntax**: Run linter or syntax check on file
- **Process**: Check if process is running
- **Port**: `netstat -ano | findstr :PORT`

Identify what user wants to check and run appropriate command.
