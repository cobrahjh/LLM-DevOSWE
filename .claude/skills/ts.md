# Test This

Run tests on recent changes or specified code.

1. Identify what was recently changed (git diff, recent edits)
2. Find relevant test files
3. Run appropriate test command:
   - Node.js: `npm test` or `npx jest <file>`
   - Python: `pytest <file>`
   - General: Check package.json scripts

4. Report results - pass/fail summary
5. If failures, show error details and suggest fixes
