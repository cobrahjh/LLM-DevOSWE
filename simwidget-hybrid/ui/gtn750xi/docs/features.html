Moving Map Display

Overview

The moving map provides real-time visualization of your aircraft position, flight plan route, and surrounding geography with multiple zoom levels and orientation modes.

Zoom Levels

Range Use Case Details Shown
2 nm Airport traffic pattern, final approach Individual taxiways, fine terrain
5 nm Departure/arrival, nearby airports Airport symbols, close navaids
10 nm Terminal area navigation Multiple airports, VORs
25 nm Short cross-country legs Regional view, major navaids
50 nm Medium cross-country Route overview, airports along route
100 nm Long cross-country, planning Major airports only
200 nm Strategic overview Large-scale route visualization

Orientation Modes

North Up

Icon: ‚¨ÜÔ∏è N

Map fixed with north at top. Aircraft symbol rotates to show heading. Traditional paper chart orientation.

Best for: Strategic planning, comparing with sectional charts

Track Up

Icon: ‚Üë TRK

Map rotates so your ground track is always pointing up. Compass rose rotates around map.

Best for: Enroute navigation, wind correction visualization

Heading Up

Icon: ‚Üë HDG

Map rotates to match magnetic heading. Shows where nose is pointing regardless of wind drift.

Map Elements

  • Aircraft Symbol: White chevron showing position and heading
  • Range Rings: Concentric circles at 25%, 50%, 75%, 100% of selected range
  • Compass Rose: N/S/E/W labels (rotates in Track/Heading modes)
  • Flight Plan Route: Magenta line connecting waypoints
  • Active Waypoint: Magenta circle highlight
  • Desired Track Line: Magenta line from aircraft to active waypoint
  • Waypoint Labels: ICAO ident and distance/ETE

Controls

  • Zoom In/Out: +/- buttons or keyboard +/-
  • Center Map: Returns panning to ownship (C key)
  • Cycle Orientation: Click orientation icon or soft key
  • Pan Map: Click and drag (touch: swipe)
  • Pinch Zoom: Two-finger pinch on touch devices

Flight Planning (FPL Page)

Creating a Flight Plan

  1. Press FPL button to open Flight Plan page
  2. Click + Add Waypoint button
  3. Enter ICAO code (e.g., KSEA, PANC, DEN) or user waypoint name
  4. Waypoint appears in route list with distance and bearing
  5. Repeat to build complete route
  6. Press Activate to begin navigation

CDI Navigation

Overview

The Course Deviation Indicator (CDI) shows lateral deviation from your desired course with automatic sensitivity scaling based on flight phase.

CDI Sources

GPS Track to active waypoint in flight plan Default mode
NAV1 VOR or ILS navigation via NAV1 radio Requires tuned frequency
NAV2 Secondary VOR navigation Backup navigation source

CDI Sensitivity Modes

ENR (Enroute)

Full Scale: ¬±5.0 nm

Active: >30nm from destination

Wide needle movement for general navigation. Each dot represents ~1nm deviation.

TERM (Terminal)

Full Scale: ¬±1.0 nm

Active: Within 30nm of destination

Tighter sensitivity for terminal area procedures. 5x more sensitive than ENR.

APR (Approach)

Full Scale: ¬±0.3 nm

Active: Within 2nm with approach loaded

Maximum precision for approach navigation. 16x more sensitive than ENR. Pulsing indicator.

CDI Display Elements

  • Needle: Shows left/right deviation from course
  • Dots: 5 dots each side (full deflection at 5 dots)
  • TO/FROM Flag: Indicates if flying to or from waypoint
  • Course: Desired track (DTK) in degrees
  • Cross Track Error: Distance left/right of course (nm)
  • Source Indicator: GPS/NAV1/NAV2 badge
  • Mode Badge: ENR/TERM/APR with color coding

OBS (Omni-Bearing Selector)

In NAV1/NAV2 modes, manually select course to/from VOR:

  1. Switch CDI source to NAV1 or NAV2
  2. Click OBS knob or soft key
  3. Rotate to select desired radial (000-359¬∞)
  4. CDI shows deviation from selected course

Direct-To Navigation

Quick Navigation

Direct-To provides instant navigation to any waypoint without modifying your stored flight plan.

Activation Methods

  1. Soft Key Method:
    • Press D‚Üí soft key
    • Enter waypoint identifier
    • Press ACTIVATE
  2. Keyboard Shortcut: Ctrl+D
  3. Map Context Menu: Right-click waypoint ‚Üí Direct-To
  4. Flight Plan: Click waypoint in FPL page ‚Üí Direct-To button

Direct-To vs Flight Plan

Feature Direct-To Flight Plan
Activation Instant (2 clicks) Build route first
Waypoint Sequence Single waypoint only Multiple waypoints
Stored Route Unchanged (temporary) Saved in FPL
Auto-Sequencing Ends at destination Continues through route
Use Case Quick diversion, practice Planned cross-country

Canceling Direct-To

  • Press D‚Üí soft key ‚Üí CANCEL
  • Activate a different Direct-To
  • Activate stored flight plan

Nearest Database (NRST Page)

Overview

Find nearby airports, VORs, NDBs, and intersections sorted by distance from your current position. Critical for emergency diversions.

Database Categories

Airports (APT)

Shows 10 closest airports with:

  • ICAO identifier
  • Distance and bearing
  • Longest runway length
  • Elevation
  • Fuel availability

VORs

VOR, VORTAC, and TACAN stations:

  • Station identifier
  • Frequency
  • Distance and bearing
  • Type (VOR/VORTAC/TACAN)

NDBs

Non-directional beacons:

  • Station identifier
  • Frequency (kHz)
  • Distance and bearing

Fixes (INT)

Named intersections:

  • Fix identifier (5 letters)
  • Distance and bearing
  • Associated airways

Quick Actions

  • Direct-To: Click waypoint ‚Üí D‚Üí button
  • Add to Flight Plan: Click waypoint ‚Üí ADD TO FPL
  • View Details: Click waypoint for full information page
  • Tune Frequency: (VOR/NDB) Click ‚Üí TUNE NAV1/NAV2

Emergency Use

In an emergency, NRST page provides fastest access to diversion airports:

  1. Press NRST button
  2. Airports sorted by distance (closest first)
  3. Verify runway length adequate for your aircraft
  4. Click airport ‚Üí D‚Üí to navigate direct

Map Overlays

Available Overlays

Toggle map overlays via soft keys or settings menu. Multiple overlays can be active simultaneously.

Terrain

Elevation-based color coding:

  • Yellow: 1,000-3,000ft
  • Orange: 3,000-6,000ft
  • Red: 6,000-10,000ft

Radio Management

Dual COM/NAV Display

GTN750 displays and controls active and standby frequencies for COM1, COM2, NAV1, and NAV2.

Frequency Tuning

  1. Click on standby frequency field
  2. Enter new frequency via keypad or knob
  3. Click SWAP or press ‚Üî to make active

Quick Tune

Airport frequencies auto-populate from NavDB:

  • View airport details page
  • Click frequency (ATIS, Tower, Ground, etc.)
  • Frequency loaded into standby
  • Swap to make active

Frequency Ranges

COM 118.000 - 136.975 MHz 8.33 kHz spacing (25 kHz legacy)
NAV 108.000 - 117.950 MHz 50 kHz spacing

Transponder

Squawk Code Entry

  1. Click transponder code display in status bar
  2. Octal keypad appears (0-7 only)
  3. Enter 4-digit code
  4. Press ENTER to confirm

Mode Selection

STBY Standby - Transponder powered but not transmitting
ON Mode A - Transmits squawk code only
ALT Mode C - Transmits code + altitude (required above 10,000ft)
GND Ground mode - Low power for surface operations

IDENT Function

Press IDENT when requested by ATC. Indicator pulses for 18 seconds.

VFR Quick-Set

Press VFR button to instantly set code to 1200.

TCAS Display

Traffic Symbols

  • Yellow Triangle: Proximate traffic (within 10nm)
  • Red Square: Collision threat (<30 seconds)
  • White Diamond: Non-threat traffic

Altitude Display

Relative altitude shown next to each target:

  • +1200: 1,200ft above you
  • -500: 500ft below you
  • 0: Same altitude (¬±100ft)

Alerts

Audio and visual warnings for collision threats:

  • Traffic Advisory (TA): Yellow, 40-second warning
  • Resolution Advisory (RA): Red, 30-second warning with climb/descend guidance
