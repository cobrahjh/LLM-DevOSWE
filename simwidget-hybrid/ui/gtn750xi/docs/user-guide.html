Interface Overview

Main Display Areas

  • Header Bar - Connection status, GPS info, time
  • Data Fields - 4 customizable corner fields (GS, TRK, ALT, etc.)
  • Map Canvas - Main moving map display
  • CDI Strip - Course Deviation Indicator with OBS/NAV source
  • Page Buttons - MAP, FPL, NRST, TAXI, PROC, AUX, CHARTS, SYSTEM
  • Soft Keys - 12 context-sensitive buttons at bottom

Connection Status

The connection indicator in the header shows:

  • ● Green - Connected to simulator
  • ● Yellow - Connecting/reconnecting
  • ● Red - Disconnected

Flight Planning

Creating a Flight Plan

  1. Press FPL page button
  2. Click + Add Waypoint button
  3. Type waypoint identifier (e.g., KSEA)
  4. Press Enter to add to flight plan
  5. Repeat for all waypoints in your route

Direct-To Navigation

Quick navigation to any waypoint:

  1. Press D→ soft key (or keyboard D)
  2. Type waypoint identifier
  3. Press ACTIVATE button
  4. Map automatically switches to MAP page

Editing Flight Plan

  • Remove Waypoint - Click waypoint, press REMOVE
  • Insert Waypoint - Click between waypoints, add new one
  • Clear All - FPL Menu → Clear Flight Plan

Auto-Sequencing

The GTN750 automatically advances to the next waypoint when:

  • Within 0.5nm of current waypoint
  • OBS mode is disabled
  • Flight plan has remaining legs

CDI & Navigation

Course Deviation Indicator

The CDI shows lateral deviation from desired course:

  • Needle Position - ±10° scale, centered when on course
  • TO/FROM - Arrow indicates direction to waypoint
  • XTK Display - Cross-track error in nautical miles
  • DTK - Desired track to waypoint
  • BRG - Actual bearing to waypoint

Navigation Sources

Click NAV source indicator to change:

  • GPS - Follow flight plan (default)
  • NAV1 - Follow VOR/ILS on NAV1 frequency
  • NAV2 - Follow VOR/ILS on NAV2 frequency

OBS Mode

Manual course selection:

  1. Press OBS soft key (or keyboard O)
  2. CDI freezes on current waypoint
  3. Turn OBS knob to set desired course
  4. Press OBS again to resume auto-sequencing

Radio Management

COM Frequencies

COM1 and COM2 each have:

  • Active frequency (currently in use)
  • Standby frequency (ready to swap)

To tune:

  1. Click standby frequency
  2. Type new frequency (e.g., 124.85)
  3. Click ⇄ button to swap active/standby

NAV Frequencies

Same procedure as COM frequencies. NAV radios provide:

  • VOR/ILS navigation signals
  • DME distance information
  • Radial/course guidance

Transponder

Set squawk code:

  1. Click XPDR code display
  2. Type 4-digit code (e.g., 1200)
  3. Press Enter

Modes: OFF, STBY, ON, ALT

Pages Overview

MAP Page

Moving map display with route visualization

Soft Keys:

  • TOPO - Toggle terrain overlay
  • TFC - Toggle traffic overlay
  • WX - Toggle weather overlay
  • D→ - Direct-To dialog
  • MENU - Map options

FPL (Flight Plan) Page

Full flight plan editor with waypoint list

Features:

  • Add/remove waypoints
  • View distance and bearing to each leg
  • Estimated time enroute
  • Total route distance

NRST (Nearest) Page

Find nearby navigation points

Tabs:

  • APT - Nearest airports with runway info
  • VOR - VOR stations with frequency
  • NDB - NDB beacons
  • INT - Intersections

TAXI (SafeTaxi) Page

Airport surface diagrams with real-time position

Controls:

  • LOAD - Load airport diagram
  • CENTER - Center on ownship
  • ZOOM+/- - Zoom in/out
  • AUTO - Auto-scale diagram
  • FOLLOW - Auto-follow mode
  • TRK UP - Track-up orientation

See SafeTaxi Guide for details

Tips & Best Practices

Flight Planning Tips

  • Use Direct-To for quick route changes in flight
  • Enable OBS mode when practicing VOR intercepts
  • Cross-fill routes from SimBrief or FlightPlan widget
  • Monitor XTK to stay on course

Map Usage Tips

  • Use terrain overlay when flying in mountainous areas
  • Enable traffic overlay for busy airspace awareness
  • Track Up mode is best for en-route navigation
  • North Up mode is best for approach planning

Performance Tips

  • Disable unused overlays to improve performance
  • Lower declutter level in busy areas
  • Use appropriate zoom level for phase of flight
