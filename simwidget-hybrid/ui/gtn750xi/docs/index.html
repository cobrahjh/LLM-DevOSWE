Overview

The GTN 750 is a comprehensive GPS navigation system emulator that provides flight planning, moving map display, navigation aids, and airport information for Microsoft Flight Simulator.

Key Capabilities

  • Multi-resolution Moving Map - 7 zoom levels from 2nm to 200nm
  • Flight Planning - Full route management with auto-sequencing
  • CDI Navigation - GPS, NAV1, NAV2 modes with OBS
  • SafeTaxi Diagrams - Real-time airport surface positioning
  • Radio Management - COM, NAV, and Transponder control
  • Database Integration - Airports, VORs, NDBs, intersections

Getting Started

Quick Start Guide

  1. Access the Widget

    Navigate to http://localhost:8080/ui/gtn750/

  2. Verify Connection

    Check for green connection indicator in the header

  3. Create Your First Route

    Press FPL button → Click + Add Waypoint → Enter ICAO code

  4. Navigate Direct-To

    Press D→ soft key → Enter waypoint → Press ACTIVATE

Features Overview

Navigation & Flight Planning

  • Moving Map Display - Multi-resolution with 3 orientation modes
  • Flight Planning - Route management with auto-sequencing
  • CDI Navigation - GPS, NAV1, NAV2 modes with OBS
  • Direct-To - Instant navigation to any waypoint
  • Nearest Database - Find nearby airports and navaids

Display & Interface

  • Data Fields - 4 customizable corner displays with 12 field types
  • Map Overlays - Terrain, Traffic, Weather layers
  • SafeTaxi Diagrams - Real-time airport surface positioning
  • Soft Keys - Context-sensitive button functions

Radio & Systems

  • Radio Management - Dual COM/NAV frequency control
  • Transponder - 4-digit code with mode selection
  • TCAS Display - Traffic alerts and visualization

API Reference

The GTN750 provides several APIs for integration:

BroadcastChannel API

Cross-widget communication via SimGlass-sync channel:

// Route updates
{
    type: 'route-update',
    waypoints: [{ident, lat, lon, altitude}, ...],
    source: 'gtn750'
}

// Direct-To activation
{
    type: 'direct-to',
    waypoint: {ident, lat, lon},
    source: 'gtn750'
}

Module APIs

  • GTNCore API - Utility functions
  • FlightPlan API - Route management
  • CDI API - Navigation functions
  • Map Renderer API - Canvas rendering
