Keyboard Tips

Efficiency Tips

  • Use D for quick Direct-To instead of navigating to FPL page
  • Press Space frequently to re-center the map on your position
  • Use number keys 1-3 to quickly switch CDI sources
  • Learn page shortcuts (M, F, N, T) to avoid clicking

Mouse Shortcuts

  • Double-click map - Quick re-center
  • Mouse wheel on map - Smooth zoom control
  • Click data field - Customize field type
  • Right-click waypoint - Quick menu (future)

Customization

You can customize keyboard shortcuts in the SYSTEM page (future enhancement). Current shortcuts follow standard Garmin conventions.
